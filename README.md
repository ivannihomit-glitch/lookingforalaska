# В поисках Аляски - Фан-сайт

Фан-сайт, посвященный сериалу "В поисках Аляски" (Looking for Alaska) по мотивам романа Джона Грина.

## Страницы сайта

1. **Персонажи** - Описание главных героев сериала
2. **Все серии** - Эпизоды с кратким описанием
3. **Актёры и роли** - Информация об актёрском составе
4. **Галерея кадров** - Фотогалерея из сериала

## Особенности

- Адаптивный дизайн для мобильных устройств
- Плавная навигация между страницами
- Анимация переходов
- Sticky navigation bar
